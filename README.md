# 📄 PA Prompt Library

Libreria di prompt per consulenti di **Public Affairs**, lobbying e comunicazione istituzionale.
Sviluppata e utilizzata quotidianamente in contesti di advocacy su dossier energetici, parlamentari e geopolitici.

---

## Struttura

```
pa-prompt-library/
├── 01_monitoring/          # Monitoraggio parlamentare e normativo
├── 02_drafting/            # Redazione documenti strategici e policy briefs
├── 03_stakeholder_mapping/ # Mappatura stakeholder e analisi relazionale
├── 04_geopolitical/        # Analisi geopolitica ed energetica
├── 05_media_analysis/      # Rassegna stampa e analisi media
├── 06_speechwriting/       # Discorsi, audizioni, posizioni pubbliche
├── 07_translation/         # Adattamento e traduzione documenti policy
├── 08_intelligence/        # Briefing di intelligence su dossier
└── 09_reporting/           # Report periodici per clienti e stakeholder
```

---

## Prompt inclusi

### 🔍 Monitoring
- Monitoraggio iter legislativo (Camera/Senato)
- Alert normativo EU (OJEU, Commissione, PE)
- Sintesi audizioni e commissioni parlamentari

### ✍️ Drafting
- Policy brief su regolamenti europei
- Leave-behind per incontri istituzionali
- Position paper e note strategiche

### 🗺️ Stakeholder Mapping
- Analisi posizionamento attori su un dossier
- Mappatura forze parlamentari su un disegno di legge
- Identificazione alleati e oppositori

### 🌍 Geopolitical Analysis
- Briefing energetico (gas, LNG, rinnovabili)
- Analisi corridoi infrastrutturali (TAP, Transmed, Greenstream)
- Scenario planning su crisi geopolitiche

---

## Come usarli

I prompt sono ottimizzati per **Claude (Anthropic)** ma adattabili a qualsiasi LLM.
Ogni file segue il formato:

```
# [Nome Prompt]
## Contesto
## Istruzioni
## Input richiesto
## Output atteso
## Note
```
